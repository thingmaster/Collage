Collage Editor

Photo collage layout tool for print-ready compositions

Getting Started

  1. Add photos — Click + Photo in the sidebar, use File > Load Photos, or drag images onto the canvas.
  2. Generate a layout — Use Layout > Generate Layout (uniform grid) or Layout > Generate Random Layout (varied sizes).
  3. Edit — Click cells to select, drag to pan, scroll to zoom, resize with corner handles.
  4. Export — Use File > Export JPEG to save your collage at full DPI.
You can also paste images from your clipboard with Ctrl+V — they appear as overlays.

Three Object Types

B Border Objects

Drawn first (behind everything). Use for backgrounds, frames, and mattes.

  • Click + Border to add a full-page border element
  • Set fill to Solid Color or Image File in properties
  • Check "Fill layout area too" to use as a full background image
  • Each border has an inner rect (auto from margins, or custom) that defines the layout area
  • Drop an image onto the "Drop background image here" zone in properties

C Cell Objects (Photos)

The main photo content, arranged by the layout engine.

  • Load photos via + Photo, File > Load Photos, or drag-drop
  • Click Layout > Generate Layout or Fixed Size Layout to arrange
  • Subject to layout automation (Generate, Randomize, Refresh)
  • Each cell has zoom, pan, position, size, and lock controls

OV Overlay Objects

Creative elements drawn on top of everything. Text, artwork, decorations.

  • Click + Text to create a text overlay with live preview
  • Ctrl+V to paste from clipboard (images, Word art, etc.)
  • Not affected by layout automation — manually positioned
  • Support pan and zoom within their bounds (drag to pan, scroll to zoom)

Canvas Mode Bar

Object Visibility

Three toggle buttons: Photo Text Border

Click to show/hide each group. Object counts shown in parentheses. Hidden groups can't be clicked or edited on the canvas.

Content Mode (default)

Dropdown with three sub-modes:

  • Select / Edit — Click to select, drag to pan, scroll to zoom, corner handles to resize
  • Insert — Drop images to replace or insert (5-zone: replace, above, below, left, right)
  • Edge Resize — Drag shared edges between adjacent cells

Align Mode

Multi-select cells and snap their edges:

  • Click cells to add to selection (purple dashed outline)
  • Shift+Click to toggle a cell in/out of selection
  • ⬆T snap tops, ⬇B snap bottoms, ⬅L snap lefts, ➡R snap rights
  • Arrow keys move the entire group

Size Mode

Multi-select cells and equalize dimensions:

  • Same selection behavior as Align mode
  • ↕H make all selected cells the same height (averaged)
  • ↔W make all selected cells the same width (averaged)

Scale Mode

Adjust the layout area and output size:

  • Drag the blue layout box edges to adjust individual margins
  • Drag inside the box to move the entire layout area (asymmetric positioning)
  • Cells rescale proportionally in real-time as you drag
  • Output size dropdown for paper presets, slider for proportional scaling

Properties Panel

The Properties panel in the sidebar adapts to the selected object type:

Cell Properties

  • File info (name, date, time, resolution, cell size)
  • Zoom slider and numeric input
  • Fix Position — lock to prevent move/resize
  • Raise to Top / Send to Back — draw order
  • Reset Crop / Add to Layout / Remove
  • → Overlay — convert cell to overlay

Overlay Properties

  • Source image info
  • Fix Position lock
  • Position (x, y) and Size (w, h) in inches
  • Mode: Transparent or Filled (with color picker)
  • Export Source PNG — save the raw source bitmap for inspection
  • → Cell — convert overlay to cell

Border Properties

  • Fix Position lock
  • Position and Size
  • Fill: Solid Color (with opacity slider) or Image File
  • Drop zone for background images
  • Fill layout area too — makes the border a full background (exclusive, only one at a time)
  • Inner rect: Auto (from margins) or Custom (defines layout bounding box)

Objects List

The sidebar Objects list shows all items in one unified view with type badges:

  • B Border objects (drawn first)
  • C Cell objects (photos, in draw order)
  • OV Overlay objects (drawn on top)
  • — Unassigned images (not in layout)

Filter buttons: All | Cells | Overlays | Borders

Each item has:

  • ▲/▼ reorder arrows (left side) — change draw order
  • Checkbox — uncheck to remove from layout (re-check to restore)
  • Thumbnail with hover preview (150px popup)
  • Click to select and show properties (green flash on canvas)

Text Overlay Creator

  1. Click + Text in the sidebar
  2. Type your text in the editor (multi-line supported)
  3. Choose font, size, color, bold, italic
  4. A 3x live preview appears centered on the canvas
  5. Click Create Overlay to finalize as a transparent PNG overlay
  6. Click Cancel to exit without creating
The preview shows at 3x actual size so you can see font details. The final overlay will be smaller.

Save & Share

Save Project (.collage)

Saves everything as a single JSON file with images embedded as base64. Large files but fully self-contained.

Share Project (.zip)

Creates a compressed ZIP with:

  • images/ folder with separate image files
  • project.collage JSON referencing images by filename

To open a shared project: File > Load Project and select the .zip file.

Export JPEG

Exports the collage at the configured DPI and paper size. No UI chrome (selection outlines, handles) in the export. Scale mode slider affects export dimensions.

Tips & Tricks

Background image: Click + Border, drop your background image, check "Fill layout area too". Use Scale mode to drag the layout box inward for a frame effect.
Can't select an overlay? Click it in the Objects list instead — guaranteed selection regardless of overlap.
Fix Position: Check this on any element to lock it from accidental moves. You can still select it for inspection.
Identify: Click Identify (or use View menu) to flash numbers on all elements for 5 seconds — match to the Objects list.
Word art artifacts: If pasted overlay text has faint lines, set Edge Trim to 1-2px in Settings.
Large projects: Save frequently. The browser has no auto-save. Use Share Project for portable backup.
